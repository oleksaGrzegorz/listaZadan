## Overview
This is a simple, responsive task list web application built with HTML, CSS, and JavaScript. The application allows users to add, mark as complete, and delete tasks.

## Features
- Add new tasks
- Mark tasks as done
- Delete tasks
- Responsive design
- Clean, minimalist interface

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Lato)

## How to Use
1. Type a task in the input field
2. Click "Dodaj zadanie" (Add Task) button or press Enter
3. To mark a task as done, click the green button
4. To delete a task, click the trash bin button

## Responsive Design
The application is fully responsive and works well on both desktop and mobile devices.

## Key Functionalities
- Tasks are stored in the browser's memory
- Instant task addition and deletion
- Green button indicates task completion
- Input field automatically focuses after task addition
